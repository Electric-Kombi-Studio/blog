Nossos Projetos

Turing Nobel Mini Games


Este é um projeto desenvolvido por nossa equipe para realizar divulgação científica de uma forma divertida, por meio de jogos criados utilizando o Scratch. São mini games que trazem, de forma lúdica e dinâmica, a história de alguns ganhadores do Prêmio Turing e do Prêmio Nobel.

Oficina de Arduino EKS

A nossa Oficina de Arduino é um projeto para ensinar lógica de programação até mesmo para quem não possui nenhuma experiência. São várias etapas de aprendizado, baseadas em tarefas simples de programação de Arduino, um computador que cabe na palma da sua mão e que apresente possibilidades infinitas de projetos de automação, robótica, internet das coisas, entre outros.
